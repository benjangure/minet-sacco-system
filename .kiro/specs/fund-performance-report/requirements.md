# Fund Performance Report - Requirements

## Overview
Track performance of each fund to understand fund health and member participation.

## User Stories
- US1: View balance and activity for each fund
- US2: Compare fund performance
- US3: Identify fastest growing funds
- US4: Export to Excel/PDF

## Functional Requirements
- FR1: Show fund balances and member counts
- FR2: Calculate contributions and withdrawals
- FR3: Show growth metrics
- FR4: Compare funds
- FR5: Export functionality

## Success Criteria
- All funds included
- Calculations accurate
- Report generates within 5 seconds
