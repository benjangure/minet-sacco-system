# Loan Portfolio Report - Requirements

## Overview
Provide comprehensive view of loan portfolio by status and product.

## User Stories
- US1: View loan count and amount by status
- US2: Identify portfolio health metrics
- US3: Breakdown by loan product
- US4: Export to Excel/PDF

## Functional Requirements
- FR1: Show loans by status (pending, active, repaid, defaulted, conditional)
- FR2: Calculate portfolio health percentage
- FR3: Show breakdown by loan product
- FR4: Display member count per status
- FR5: Export functionality

## Success Criteria
- All loans correctly categorized
- Portfolio health calculation accurate
- Report generates within 5 seconds
