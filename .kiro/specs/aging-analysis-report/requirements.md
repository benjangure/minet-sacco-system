# Aging Analysis Report - Requirements

## Overview
Categorize loans by delinquency status to identify problem loans and collection priorities.

## User Stories
- US1: View loans by age of delinquency
- US2: Identify loans requiring immediate action (90+ days)
- US3: Export aging analysis to Excel/PDF
- US4: Filter by loan product or member

## Functional Requirements
- FR1: Categorize loans into 4 age brackets (0-30, 31-60, 61-90, 90+ days)
- FR2: Calculate total overdue amount and percentage
- FR3: Show member details for each overdue loan
- FR4: Export functionality
- FR5: Access control (ADMIN, AUDITOR, TREASURER)

## Success Criteria
- All overdue loans are correctly categorized
- Calculations are accurate
- Report generates within 5 seconds
