# Delinquency Report - Requirements

## Overview
Identify members with overdue payments for collection activities.

## User Stories
- US1: View all delinquent members
- US2: Sort by collection priority
- US3: Filter by days overdue
- US4: Export to Excel/PDF

## Functional Requirements
- FR1: List all delinquent members with details
- FR2: Calculate days overdue
- FR3: Assign collection priority
- FR4: Show total delinquent amount
- FR5: Export functionality

## Success Criteria
- All delinquent members identified
- Days overdue calculated correctly
- Priority assignment accurate
- Report generates within 5 seconds
